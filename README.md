# Credit Card Fraud Detection Project

## Proje Açıklaması

Bu projede kredi kartı dolandırıcılığı tespiti için makine öğrenmesi modelleri ve istatistiksel analiz yöntemleri kullanılmıştır.

## Kullanılan Modeller

- Logistic Regression
- Decision Tree
- Random Forest

## Kullanılan İstatistiksel Yöntemler

- Bernoulli Distribution
- Binomial Distribution
- Confidence Interval
- Hypothesis Testing
- Cross Validation

## Veri Seti

Kaggle Credit Card Fraud Dataset:

https://www.kaggle.com/datasets/mlg-ulb/creditcardfraud
